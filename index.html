Viz Grimoire project

Viz Grimoire (usually pronounced in the French way) is a framework to visualize software analytics based in metrics for data from repositories related to software development: source code management (aka version control) and issue tracking (aka bug reporting) systems, mailing lists, etc.

Viz Grimoire tools supports now Metrics Grimoire (CVSAnalY, Bicho and MailingListStats) data.

Viz Grimoire has its origins in the Bitergia company, a spin-off company of LibreSoft research providing professional services around Software Analytics.

Source code, reporting bugs, communication

There are several resources that can be of use for getting the source code, contributing, reporting bugs, or just communicating with developers:

  • IRC channel: we usually hang out at #metrics-grimoire in freenode (access via webchat)
  • Mailing list: metrics-grimoire at lists.libresoft.es, for developers, contributors and users (archives are available)
  • Source code: available in several git repositories at the VizGrimoire GitHub project
  • Submitting bug reports: we use the GitHub issue tracking system ("Issues" tab in each of the repositories of the VizGrimoire GitHub project
